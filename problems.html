A selection of typical CSP problems

Knights problem

How can a knight jump on an NxN chessboard in such a way that it visits every square exactly once? Consider two cases:
  1. The knight's tour ends on the same square where it begun
  2. The knight's tour ends on a dfferent square

  3. In the house immediately on the right of that housing the wolf, they smoke Luky Strike.
  4. The Spaniard smokes Kent.

Maps colouring problem

Is there a way to color a map using N colors such that no adjacent countries do have the same color? Fnd such a map coloring.

Crosswords

Read an empty crosswords square (i.e. an MxN square and the positions of the black cells). And read a dictionary of words. Fill the crosswords square, such that all words formed on lines and columns appear in the dictionary.

Magic square

Fill an NxN square with all numbers from 1 to N^2 such that the sum of all elements on any line, any column and any diagonal are all the same.

Sudoku

Solve a sudoku game.
  1. A 9x9 board, divided in 9 quares of 3x3.
  2. Each cell has a number from 1 to 9.
  3. Any line has all numbers exactly once.
  4. Any column has all numbers exactly once.
  5. Any square has all numbers exactly once.
The board is read partially filled. It has to be filled completely.

Timetable

The following items are available:
  • A list of classes;
  • A list of classrooms;
  • A list of tutors, and the list of subjects they can teach;
  • Lists of subjects to be taught to classes;
Decide on the classes timetable (i.e. who teaches what, when, where and to whom).

Car sequencing

The car sequencing problem involves scheduling the assembly of cars which are distinguished into classes by different sets of options. The assembly line has different stations where the various options (air-conditioning, sun-roof, radio) are installed by teams. Each such station has a certain workload capacity given by the parameters M and N, denoting M out of a sequence of N cars can be fitted the given option. The objective is to produce such a sequence of cars that (i) the demand for each car class is satisfied and (ii) none of the capacity constraints is violated.

Job-shop scheduling

The following are given:
  • A finite set of n jobs
  • Each job i consists of a chain of n_i operations
  • A finite set of m machines
  • Each machine can handle at most one operation at a time
  • Each operation needs to be processed during an uninterrupted period of a given length on a given machine
The purpose is to find a schedule, that is, an allocation of the operations to time intervals to machines, that has minimal length

Balanced Academic Curriculum

Design a balanced academic curriculum by assigning periods to courses in a way that the academic load of each period is balanced, i.e., as similar as possible. The curriculum must obey the following administrative and academic regulations:
  1. Academic curriculum: an academic curriculum is defined by a set of courses and a set of prerequisite relationships among them.
  2. Number of periods: courses must be assigned within a maximum number of academic periods.
  3. Academic load: each course has associated a number of credits or units that represent the academic effort required to successfully follow it.
  4. Prerequisites: some courses can have other courses as prerequisites.
  5. Minimum academic load: a minimum amount of academic credits per period is required to consider a student as full time.
  6. Maximum academic load: a maximum amount of academic credits per period is allowed in order to avoid overload.
  7. Minimum number of courses: a minimum number of courses per period is required to consider a student as full time.
  8. Maximum number of courses: a maximum number of courses per period is allowed in order to avoid overload.
The goal is to assign a period to every course in a way that the minimum and maximum academic load for each period, the minimum and maximum number of courses for each period, and the prerequisite relationships are satisfied. An optimal balanced curriculum minimises the maximum academic load for all periods. Note that we could consider other types of balance criterion such as minimising the sum of the academic load of all periods.
